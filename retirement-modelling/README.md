# Retirement Modelling Dashboard (Power BI + R)

This dashboard models personalised retirement outcomes using R and Power BI.
It includes:
- CPI forecast modelling
- Mortality estimates using Australian Life Tables
- Expense projections
- Investment return modelling
- Monte-Carlo Simulations
